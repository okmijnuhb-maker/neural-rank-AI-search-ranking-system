About NeuralRank

A senior-level search ranking system built from scratch — replicating how Google, Microsoft and Amazon rank search results using machine learning.

What is NeuralRank?

NeuralRank is a complete Learning-to-Rank (LTR) system that uses machine learning to rank search results more accurately than traditional keyword search. It was built as a placement portfolio project demonstrating end-to-end data science, machine learning, and full-stack development skills.

The system trains on the real MS MARCO dataset — the same dataset used by Microsoft to improve Bing search — and achieves a 17.5% improvement in NDCG@10 over the BM25 baseline.

Development Journey

Notebook 1 — Data Collection
Downloaded MS MARCO from Hugging Face. Cleaned 149 queries with no relevant passages. Saved 4,851 clean queries locally.
Notebook 2 — BM25 Baseline
Built BM25Okapi index on 39,962 passages. Computed NDCG@10=0.5069, MRR=0.3573, Precision@10=0.1406 as baseline targets to beat.
Notebook 3 — Feature Engineering
Engineered 23 features across 4 categories. Added LSA semantic similarity via TF-IDF + SVD pipeline. Computed feature correlation heatmap and importance analysis.
Notebook 4 — LambdaMART Training
Trained XGBoost with rank:ndcg objective. Hyperparameter tuning across 3 configs. Early stopping at round 16. Achieved NDCG@10=0.5956 — 17.5% improvement.
Notebook 5 — SHAP Explainability
Applied TreeExplainer on all 39,962 pairs. Generated beeswarm, waterfall, force, dependence and interaction plots. Saved interactive HTML report.
Notebook 6 — A/B Testing
Simulated 1,000 user sessions. Computed 20.1% CTR lift. Proved statistical significance with Mann-Whitney U (p=0.041) and Cohen's d effect size.
Notebook 7 — Evaluation Dashboard
Built interactive Plotly dashboard combining all metrics, charts and analysis. Exported as standalone HTML. Created final summary CSV.
